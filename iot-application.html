SIT210 Developing IOT 2023

1. Activity 1 Iot applications:

In this activity, the definition of IOT apps has been explained and it’s examples and the main features have been described. Moreover, IoT uses machine learning techniques to analyse massive amounts of linked sensor data in the cloud which includes the features like communication, network, security and smart parking, smart grids , smart homes are the some of the examples.

2. Acticity 2 Smart Building:

In this activity, the concept is smart building is explained. Smart building has intelligent networking and automation, independent actions and ability to learn with the features such as multifunctionality, adaptability, and security. It would have places with varying climate, and maintain temperature in accordance to the incoming real time data.

3. Activity 3 Control Interface:

In this activity, the concept of control interface is understood. It’s main features are as such as eco-efficiency, interactive UI, high security and the parts like sensors, user interface and cloud software. The users would be customers, residents, employees and visitors.
